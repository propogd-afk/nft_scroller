# Простой бот - только по цене

Самая простая версия для тебя: парсит NFT подарки Telegram только по диапазону цен, который ты сам ставишь.

## Что делает
- Ты пишешь в ТГ боту `/setprice 0.5 10` - и он ищет ВСЕ подарки от 0.5 до 10 TON
- Никаких токенов бирж вручную получать не нужно - использует твой `api_id` / `api_hash` для авто-логина в MRKT
- Фильтр по статусу тоже есть: занят / на продаже / гибрид

## Установка (2 минуты)

```bash
cd simple-price-bot
pip install -r requirements.txt
```

Открой `.env` - туда уже подставлены твои данные:
```
BOT_TOKEN=8935270608:AAEO4...
API_ID=20045757
API_HASH=7d3ea0c0d4725498789bd51a9ee02421
```

Открой `config.yaml` и поставь начальную цену:
```yaml
min_price: 0.5
max_price: 10.0
```

Запуск:
```bash
python main.py
```

## Команды в Telegram

Открой своего бота (того, чей токен ты дал) и напиши:

- `/start` - подписаться на уведомления
- `/setprice 1 25` - поменять диапазон на 1-25 TON (прямо в чате, без перезапуска!)
- `/status` - показать текущие настройки
- `/stop` - отписаться

Бот будет каждые 20 секунд проверять MRKT и кидать тебе сообщение если нашел подарок в твоей цене.

## Статусы: занят / на продаже / гибрид

В `config.yaml`:
```yaml
status_filter:
  allowed: ["on_sale", "free"]  # только на продаже и свободные

# другие варианты:
# ["occupied", "locked"] - только занятые
# ["hybrid"] - только гибриды (и аукцион и продажа)
# ["auction"] - только аукционы
```

## Безопасность ⚠️

Ты скинул токен бота и api_hash в открытом чате. Это секретные ключи!

1. После теста смени токен бота у @BotFather -> /mybots -> выбери бота -> API Token -> Revoke
2. api_id/api_hash можно не менять, но не свети

В этой папке `.env` уже с твоими данными - не заливай его на GitHub!

## Как работает

1. `amrkt` библиотека использует `api_id/api_hash` чтобы залогиниться в Telegram и получить `initData` для MRKT
2. Делает запрос к `https://api.tgmrkt.io/api/v1/gifts/saling` с фильтром `minPrice` / `maxPrice` (в наноТОНах)
3. Если gift_id уже видели - пропускает (чтобы не спамить)
4. Проверяет статус (занят/на продаже/гибрид)
5. Отправляет сообщение через твой BOT_TOKEN

Никаких ручных токенов MRKT не нужно!
